Test Reports

Explore detailed reports for each hardware and network test. Each entry describes what is being validated, making it easy to add new tests in the future.

Note: All test scripts and their requirements are available in the GitHub repository.
These tests use my own custom Universal Test Framework (UTFW) for consistent and automated validation.

Nightly Regression Suite
Comprehensive end-to-end system test covering all modules: UART communication, SNMP protocol, web interface (PCAP), EEPROM persistence, power monitoring (HLW8032), ADC/thermal sanity checks, and relay stress switching. This suite aggregates results from all individual tests and represents the overall stability of the current firmware.
Serial Communication Test Report
Verifies UART communication between controller and peripheral devices, including data integrity, error handling, and protocol compliance.
Network SNMP Test Report
Tests SNMP protocol support, ensuring correct device monitoring, trap generation, and remote management capabilities.
Network PCAP Analysis Report
Analyzes captured network packets to validate communication timing, HTTP request/response patterns, and correct data encapsulation.
PCAP Capture Test Report
Validates real-time packet capture functionality and ensures consistent capture across Ethernet sessions.
Power Monitoring Test Report (HLW8032)
Verifies power, current, and voltage measurements using the HLW8032 sensor, including accuracy and calibration consistency.
EEPROM Persistence Test Report
Checks EEPROM data integrity, verifying correct storage, read/write cycles, and recovery behavior after power loss.
Sanity Check Test Report
Performs fundamental system checks to ensure all major subsystems initialize and operate as expected after boot.
Relay Stress Switching Test Report
Executes prolonged relay switching cycles to test endurance, heating, and response stability under load.
