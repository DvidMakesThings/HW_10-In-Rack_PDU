UART command handler for ENERGIS PDU (10-Inch Rack).

Version
1.0
Date

Implements command parsing and execution over UART (USB-CDC). Provides network, system, and EEPROM control via textual commands.

◆ getSysInfo()

void getSysInfo ( void )

Print system information (core voltage, clocks, serial, firmware).

Print system information (clock frequencies, voltage).

Reads device serial number and firmware version from EEPROM (SYS_INFO area), with graceful fallbacks if data is missing or checksum fails. Then prints voltage selector, core voltage estimate, and key clock frequencies.

EEPROM SYS_INFO layout (variable-length, null-terminated strings):
[0 .. N] : Device Serial (ASCII, '\0' terminated) [N+1 .. M] : Firmware Version (ASCII, '\0' terminated) <– optional
Fallbacks:
  • If EEPROM is empty/invalid: serial <- DEFAULT_SN, fw <- FW_VERSION
  • If only serial exists: fw <- FW_VERSION
Note
Uses ECHO(...) for UART-friendly, parseable output lines. Keeps previous OK header and format style.
